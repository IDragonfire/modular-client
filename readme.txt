Nozon Project Manager
--------------------

This is the source code for the Nozon Project Manager.

Pre-requisites are :

- Python 2.6/2.7 32 bits
- PyQt4 with QT 4.7.2
- Python Win32Api


If you want to contribute back to the project, please make a fork and create pull-Requests of your changes.

IMPORTANT : Altering the code may render this lobby not functional with the server.
In case of unofficial alteration of this code, no support will be given !
